# fixtures/

Place the following image files here before running tests:

- `fish_sample.jpg`   — A clear freshwater fish photo (used in upload & AI identification tests)
- `landscape.jpg`     — A non-fish image (used in negative identification tests)
- `large_image.jpg`   — Any image larger than 5MB (used in file size validation tests)

These files are excluded from version control (.gitignore) since they may be large.
Add them manually to your local copy of the test suite.
